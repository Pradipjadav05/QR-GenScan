# qr_coed_gen_scan

A new Flutter project.

## Getting Started

Add below dependency for app:
```dart
  //used to scan QR
  scan:
  // used for generate QR
  qr_flutter:
  // for select image from gallery
  image_picker:
  // for request permission
  permission_handler:
```
